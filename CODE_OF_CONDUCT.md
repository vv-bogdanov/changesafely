# Code of conduct

We want ChangeSafely collaboration to be technically rigorous, respectful, and safe.

Contributors must communicate constructively, critique work rather than people,
respect differing experience and perspectives, and avoid harassment, threats,
discrimination, sexualized conduct, or disclosure of another person's private
information. Security reports and private repository data must remain confidential.

Maintainers may edit or remove contributions that violate these expectations and may
temporarily or permanently restrict participation. Report conduct concerns privately
to the repository owner through GitHub. Do not use a public issue when doing so would
expose personal or sensitive information.

Enforcement decisions should be proportionate, documented privately, and focused on
protecting affected participants and the project community.
